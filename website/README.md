## Tech Stack

- Next.js 15
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Phosphor Icons

## Run Locally

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

## Validate

```bash
npm run lint
npm run build
```
