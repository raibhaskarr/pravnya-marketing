# Pravnix.ai Website

Production website implementation for Pravnix Labs.

Source handbook:

```text
pravnya-marketing/docs/pravnix-website-handbook.md
```

## Development

```bash
npm install
npm run dev
```

## Validation

```bash
npm run lint
npm run build
```

## Notes

- This is the company website, not the Pravnya product website.
- The site is dark-first and uses the Pravnix design system from the handbook.
- No customer logos, metrics, screenshots, partnerships, or biographies are fabricated.
- Contact form is frontend-only until production integration is configured.
